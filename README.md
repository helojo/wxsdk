# wxsdk
微信支付、分享、联合登录SDK

Gradle:

```
compile 'com.joy.wxsdk:wxsdk:3.1.1'
```

Maven:

```
<dependency>
  <groupId>com.joy.wxsdk</groupId>
  <artifactId>wxsdk</artifactId>
  <version>3.1.1</version>
  <type>pom</type>
</dependency>
```

 Ivy:

 ```
<dependency org='com.joy.wxsdk' name='wxsdk' rev='3.1.1'>
  <artifact name='$AID' ext='pom'></artifact>
</dependency>
 ```
